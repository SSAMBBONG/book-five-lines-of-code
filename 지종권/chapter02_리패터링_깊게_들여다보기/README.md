# 2장 리팩터링 깊게 들여다보기

## 코드 개선

가독성 : 의도를 전달히기 위한 코드 성질

어떻게 ?

코딩 컨벤션 지정, 주석 달기, 변수, 메서드, 클래스 및 파일 이름 지정, 공백 사용, '함께 변하는 것은 함께 있어야 한다' 불변속성의 범위제한 등등

리팩터링 세 가지 핵심

1. 의도를 전달함으로써 가독성 향상
2. 불변속성의 범위제한을 통한 유지보수성 향상
3. 범위밖의 코드에 영향을 주지않고 1항과 2항을 수행 (예약)
